Do you know what is a JavaScript Event Loop?

This term has haunted so many developers, but is a really important concept of JavaScript.

The event loop got its name because of how it's usually implemented, which usually resembles:

while (queue.waitForMessage()) {
queue.processNextMessage()
}

queue.waitForMessage()
waits synchronously for a message to arrive (if one is not already available and waiting to be handled).
